# ⚽ Football Turf Booking System - Backend

This repository contains the **backend API** for the **Football Turf Booking System**.

The backend is developed using **Django** and **Django REST Framework (DRF)** to provide secure and scalable RESTful APIs for managing users, turf slots, and bookings.

---

## 🚀 Features

- User Registration & Authentication
- JWT-based Authentication
- Turf Slot Management
- Football Turf Booking System
- Secure REST API
- Organized RESTful API architecture

---

## 🛠️ Tech Stack

| Technology | Description |
|-----------|-------------|
| Python | Backend Programming Language |
| Django | Web Framework |
| Django REST Framework | API Development |
| SQLite / PostgreSQL | Database |
| JWT Authentication | Secure API Authentication |
